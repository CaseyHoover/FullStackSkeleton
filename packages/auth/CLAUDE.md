# Auth Package

BetterAuth with GitHub OAuth and Prisma adapter.

- `src/index.ts` — server-side auth config (used by API)
- `src/client.ts` — client-side auth helpers (used by web)
- `src/next.ts` — Next.js server function wrappers

Apps import from `@skeleton/auth`, `@skeleton/auth/client`, or `@skeleton/auth/next`.
