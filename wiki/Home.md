# FullStackSkeleton Wiki

Welcome to the FullStackSkeleton wiki.

This wiki is version-controlled alongside the source code in
[`wiki/`](https://github.com/CaseyHoover/FullStackSkeleton/tree/main/wiki).
Edit pages by opening a pull request — wiki-only PRs skip the full CI
suite, but markdown linting still runs.

## Pages

- [Home](Home.md) — this page

## Adding a page

1. Create a new `.md` file in this directory.
2. Link to it from `Home.md` (or another existing page).
3. Open a pull request.

Conventional commit messages still apply — prefix the commit and PR title
with `docs:` for wiki changes (see
[CONTRIBUTING.md](../CONTRIBUTING.md)).
